# First Experiment

Run your first FluxLoop experiment.

## Quick Start

```bash
# 1. Create project
fluxloop init project --name my-agent
cd fluxloop/my-agent

# 2. Generate inputs
fluxloop generate inputs --limit 10

# 3. Run experiment
fluxloop run experiment
```
