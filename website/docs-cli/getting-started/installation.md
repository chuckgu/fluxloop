# CLI Installation

Install the FluxLoop CLI for Python.

## Requirements

- Python 3.8 or higher
- pip

## Install via pip

```bash
pip install fluxloop-cli
```

## Verify Installation

```bash
fluxloop --version
```

You should see:
```
FluxLoop CLI version 0.2.1
```
