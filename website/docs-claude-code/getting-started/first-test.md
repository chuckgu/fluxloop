# Your First Test

Run your first FluxLoop test in Claude Code.

## Step 1: Setup

```bash
/fluxloop setup
```

This will guide you through:
1. Authentication
2. Project selection
3. Configuration

## Step 2: Run Test

```bash
/fluxloop test
```

## Step 3: View Results

Results are displayed in Claude Code and uploaded to FluxLoop Web.
