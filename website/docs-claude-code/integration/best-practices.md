# Best Practices

Tips for using FluxLoop effectively in Claude Code.

## Testing Strategy

- Use `/fluxloop test --smoke` for quick iterations
- Run `/fluxloop test` before committing
- Review detailed results in FluxLoop Web

## Team Workflow

- Share projects via FluxLoop Web
- Use consistent API keys per environment
- Review test results as a team

## Performance

- Use smoke tests during development
- Run full tests in CI/CD
- Monitor trends in FluxLoop Web
