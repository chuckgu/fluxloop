# Development Workflow

Integrate FluxLoop into your development workflow.

## Daily Development

```
1. Write code
2. Run /fluxloop test --smoke
3. Review results
4. Iterate
```

## Before Commit

```bash
/fluxloop test
```

## Team Collaboration

1. Share project in FluxLoop Web
2. Team members run `/fluxloop setup`
3. Everyone tests against same scenarios
