# SDK Installation

Install the FluxLoop SDK for Python.

## Requirements

- Python 3.8 or higher
- pip

## Install via pip

```bash
pip install fluxloop
```

## Verify Installation

```python
import fluxloop
print(fluxloop.__version__)  # Should print: 0.1.0
```
