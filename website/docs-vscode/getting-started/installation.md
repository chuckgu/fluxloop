# VSCode Extension Installation

Install the FluxLoop VSCode extension.

## From Marketplace

1. Open VSCode
2. Extensions (Cmd+Shift+X)
3. Search "FluxLoop"
4. Install

## Prerequisites

```bash
pip install fluxloop-cli fluxloop
```
